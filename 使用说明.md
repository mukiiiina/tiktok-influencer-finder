# TikTok 博主搜索 Skill - 使用与分享指南

## 这个 Skill 是什么？

这是一个适用于 Claude Code / OpenClaw 的技能插件。安装后，你只需要用自然语言告诉它你想找什么类型的 TikTok 博主，它就会自动帮你：

1. 分析你的需求，生成多角度搜索关键词（中英文双语）
2. 在多个来源搜索匹配的博主（排行网站、博主数据库、内容聚合平台等）
3. 整理成格式漂亮的 Excel + CSV 表格，按粉丝量排序、颜色分层

支持中英文输入输出，表头可以切换为中文或英文。

---

## 如何安装这个 Skill

### 前提条件

你需要有一个部署好的 Claude Code / OpenClaw 机器人。

### 方法一：从 GitHub 仓库安装（推荐，最适合分享）

把仓库克隆到 Claude Code 的 skills 目录下，机器人就能自动识别并使用：

**macOS / Linux:**
```bash
git clone https://github.com/你的用户名/tiktok-influencer-finder.git ~/.claude/skills/tiktok-influencer-finder
```

**Windows:**
```bash
git clone https://github.com/你的用户名/tiktok-influencer-finder.git %USERPROFILE%\.claude\skills\tiktok-influencer-finder
```

安装完成后重启 Claude Code 会话，Skill 就会被自动加载。

### 方法二：直接告诉你的 OpenClaw 机器人安装

在和你的 OpenClaw 机器人的对话中，直接发送：

```
请帮我把这个 GitHub 仓库克隆到 skills 目录下作为一个新 skill 安装：
https://github.com/你的用户名/tiktok-influencer-finder
```

机器人会执行类似下面的命令来完成安装：
```bash
git clone https://github.com/你的用户名/tiktok-influencer-finder.git ~/.claude/skills/tiktok-influencer-finder
```

### 方法三：手动复制文件

直接把 `tiktok-influencer-finder` 整个文件夹放到 skills 目录下：
- **Windows**: `%USERPROFILE%\.claude\skills\tiktok-influencer-finder\`
- **macOS/Linux**: `~/.claude/skills/tiktok-influencer-finder/`

确保 `SKILL.md` 文件在该目录的根部。

### 安装 Python 依赖

表格生成需要 openpyxl 库，首次使用前需要安装：

```bash
pip install openpyxl
```

## 如何使用

安装好之后，直接在 Claude Code / OpenClaw 中用自然语言说你的需求就行。机器人会自动识别你的意图并触发这个 Skill。

### 中文使用示例

```
帮我找TikTok上做AI工具推荐的博主，要有趣的风格，粉丝量不限
```

```
我想推广一款护肤产品，帮我找TikTok上的美妆博主，主要面向美国市场，粉丝10万到100万
```

```
找一些TikTok上做宠物内容的博主，要经常发猫咪视频的
```

```
帮我找TikTok上的科技数码博主，要做开箱视频的，把结果整理成中文表格
```

### English 使用示例

```
Find TikTok influencers who review fitness apps, targeting the US market
```

```
I'm launching a new food delivery app. Find me TikTok food bloggers who do reviews
```

### 进阶用法

**追加到已有表格**：
```
我已经有一份博主名单 influencer_list.xlsx，帮我再找一些游戏类博主加进去
```

**指定表格语言**：
```
帮我找美妆博主，表格用中文表头
```

## 输出说明

每次搜索完成后，Skill 会生成两个文件：

### Excel 文件 (.xlsx)
- **博主列表 Sheet**: 所有博主信息，按粉丝量从高到低排序
  - 粉色 = 头部博主 (100万+)
  - 橙色 = 中腰部博主 (10万-100万)
  - 绿色 = 小博主 (1万-10万)
  - 蓝色 = 素人博主 (<1万)
  - 主页链接可点击跳转
- **搜索摘要 Sheet**: 搜索条件、日期、各层级博主数量统计

### CSV 文件 (.csv)
- 纯文本格式，可以直接导入 Google Sheets、飞书、腾讯文档等

---

## 每位博主收集的信息

| 类别 | 字段 |
|------|------|
| 基础信息 | 博主名称、TikTok 用户名、主页链接、内容领域、粉丝数 |
| 互动数据 | 互动率、平均播放量、发布频率 |
| 商务信息 | 联系方式、预估报价、品牌合作历史 |
| 受众画像 | 受众年龄/性别/地区、内容语言 |
| 内容详情 | 内容风格、个人简介、热门内容链接、其他平台账号 |

## 常见问题

**Q: 安装后 Skill 没有被识别怎么办？**
A: 检查目录结构是否正确 -- `SKILL.md` 必须在 `~/.claude/skills/tiktok-influencer-finder/SKILL.md` 这个路径。然后重新开一个 Claude Code 会话。

**Q: 搜索出来的博主数据准确吗？**
A: 数据来自公开的博主排行网站和数据库，粉丝数和互动率可能有一定延迟。建议在联系博主前，去他们的 TikTok 主页确认最新数据。

**Q: 可以搜索除了 TikTok 之外的平台吗？**
A: 这个 Skill 专注于 TikTok 博主搜索。如果博主同时活跃在其他平台（YouTube、Instagram 等），会在"其他平台"一栏标注。

**Q: 需要付费吗？**
A: 这个 Skill 本身完全免费，MIT 开源协议。
