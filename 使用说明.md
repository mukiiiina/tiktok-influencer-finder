# TikTok 博主搜索 Skill - 使用与分享指南

## 这个 Skill 是什么？

这是一个适用于 Claude Code / OpenClaw 的技能插件。安装后，你只需要用自然语言告诉它你想找什么类型的 TikTok 博主，它就会自动帮你：

1. 分析你的需求，生成多角度搜索关键词（中英文双语）
2. 在多个来源搜索匹配的博主（排行网站、博主数据库、内容聚合平台等）
3. 整理成格式漂亮的 Excel + CSV 表格，按粉丝量排序、颜色分层

支持中英文输入输出，表头可以切换为中文或英文。

---

## 如何安装这个 Skill

### 前提条件

你需要有一个能加载 skill 的 AI 机器人（比如 Claude Code 或 OpenClaw）。

### 方法一：直接让机器人安装（最简单，推荐）

在和你的机器人的对话中，直接发送：

```
帮我安装一个TikTok博主搜索的skill，GitHub仓库地址是：
https://github.com/mukiiiina/tiktok-influencer-finder
请克隆到你的skills目录下，然后安装依赖 openpyxl
```

机器人会自动找到正确的 skills 目录并完成安装。

### 方法二：手动克隆仓库

把仓库克隆到机器人的 skills 目录下（具体路径取决于你用的平台）：

```bash
git clone https://github.com/mukiiiina/tiktok-influencer-finder.git
pip install openpyxl
```

安装完成后重启会话，Skill 就会被自动加载。

### 方法三：下载 ZIP 安装

从 GitHub 下载仓库的 ZIP 压缩包，解压后把 `tiktok-influencer-finder` 文件夹放到机器人的 skills 目录下。确保 `SKILL.md` 文件在该文件夹的根部。

### 安装 Python 依赖

表格生成需要 openpyxl 库，首次使用前需要安装：

```bash
pip install openpyxl
```

## 输出说明

每次搜索完成后，Skill 会生成两个文件：

### Excel 文件 (.xlsx)
- **博主列表 Sheet**: 所有博主信息，按粉丝量从高到低排序
  - 粉色 = 头部博主 (100万+)
  - 橙色 = 中腰部博主 (10万-100万)
  - 绿色 = 小博主 (1万-10万)
  - 蓝色 = 素人博主 (<1万)
  - 主页链接可点击跳转
- **搜索摘要 Sheet**: 搜索条件、日期、各层级博主数量统计

### CSV 文件 (.csv)
- 纯文本格式，可以直接导入 Google Sheets、飞书、腾讯文档等

## 每位博主收集的信息

| 类别 | 字段 |
|------|------|
| 基础信息 | 博主名称、TikTok 用户名、主页链接、内容领域、粉丝数 |
| 互动数据 | 互动率、平均播放量、发布频率 |
| 商务信息 | 联系方式、预估报价、品牌合作历史 |
| 受众画像 | 受众年龄/性别/地区、内容语言 |
| 内容详情 | 内容风格、个人简介、热门内容链接、其他平台账号 |

## 常见问题

**Q: 安装后 Skill 没有被识别怎么办？**
A: 检查 `SKILL.md` 文件是否在 skill 文件夹的根部。然后重新开一个会话试试。如果还不行，可以直接让机器人重新安装一次。

**Q: 搜索出来的博主数据准确吗？**
A: 数据来自公开的博主排行网站和数据库，粉丝数和互动率可能有一定延迟。建议在联系博主前，去他们的 TikTok 主页确认最新数据。

**Q: 可以搜索除了 TikTok 之外的平台吗？**
A: 这个 Skill 专注于 TikTok 博主搜索。如果博主同时活跃在其他平台（YouTube、Instagram 等），会在"其他平台"一栏标注。

**Q: 需要付费吗？**
A: 这个 Skill 本身完全免费，MIT 开源协议。
